About

 Youtube-DL GUI is a Graphical User Interface made for youtube-dl and created by me, Guilherme Francisco de Freitas. It was created using C# over a month and still receives updates. It's completely free and open source, and will remain as such forever.

Features

  • Free!
  • Open source
  • Thumbnail previewer
  • Video list importing
  • Compatible with hundreds of streaming services
  • Low resource usage
  • Easy to use
  • Accepts numerous video and audio formats
  • Updates youtube-dl automatically
  • Video queueing

Compatible Websites

 You can check which formats are compatible at youtube-dl's website

Compatible Formats

  • mp4
  • flv
  • 3gp
  • webm
  • avi
  • mp3
  • m4a
  • flac
